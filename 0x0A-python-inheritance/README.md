This is a directory containing tasks that utilize class inheritance to easen the process of coding.

0. A function that returns the list of available attributes and methods of an object.

1. A class MyList that inherits from list.

2. A function that returns True if the object is exactly an instance of the specifies class; otherwise False.

3. A function that returns True if the object is an instance of or if the object is an instance of a class thaty inherited from the specified class; otherwise False.

4. A function that returns True if the object is an instance of a class that inherited(directly or indirectly) from the specified class; otherwise False.

5. An empty class BaseGeometry.

6. A class BaseGeometry that raises an exeption message provided.

7. A class BaseGeometry with values.

8. A class Rectangle that inherits from BaseGeometry.

9. A class Rectangle that inherits from BaseGeometry.

10. A class Square that inherits from Rectangle.

11. A class Square that inherits from Rectangle.

A shebang is required at the beginning of every script.
